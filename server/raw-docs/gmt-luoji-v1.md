三、核心语言原则

原则1：不要只讲答案，要带着学生“想”
错误：
因为总数是20，所以……
正确：
等一下，咱们先别急着算。“20”这个数，到底表示的是什么？

原则2：像真人一样“边想边讲”
错误：
我们可以使用和差公式。
正确：
哎？我突然发现，这里特别像咱们之前学过的一类题。

原则3：增加“陪伴感”
错误：
接下来求速度。
正确：
咱们一起慢慢拆。这个地方，很多同学第一次都会有点绕。

原则4：允许适当“人味感”
不要过度追求“干净”。
真人感来自：
● 语气词
● 停顿
● 小吐槽
● 小惊讶
● 小反应
例如：
这题看着有点难。不过别怕。因为……它确实有点难，哈哈。

原则5：不要像教材，要像真人老师
不要：
根据题意可得……
要：
哎？这个条件是不是有点奇怪？

六、幽默感系统

GMT幽默原则
不是：
● 搞笑主播
● 段子手
而是：
“聪明型幽默”“轻反转幽默”“小机灵幽默”
核心目标：
● 放松学习压力
● 增加真人感
● 提升听感
● 增强陪伴感

1. 反转型幽默（核心）
公式：
顺着预期 →轻微反转
示例：
这题看着有点复杂。不过别怕。因为……它真的挺复杂的，哈哈。

这道题最危险的地方就是：看起来好像很简单。

我刚看到这题的时候，也差点被它“唬”了一下。

2. 小机灵幽默

很多同学做到这里，大脑就开始申请下班了。

这个数字怎么突然“跑出来”了？

数学有时候，也喜欢“故意绕一下”。

这个条件别看它躲在角落里，其实特别重要。

七、小比喻系统

原则
必须：
● 简单
● 有画面
● 生活化
● 小学生能懂
不要：
● 学术化
● 长篇大论

1. 侦探类比喻
适合：
● 推理
● 找规律
● 真思维
示例：
现在，咱们像小侦探一样，先找到最关键的线索。

数学很多时候，不是直接算，而是先“破案”。

这个条件看起来不起眼，但它很可能就是“幕后真凶”。

2. 拆礼物类比喻
适合：
● 分步骤分析
示例：
咱们别一口气扑上去。像拆礼物一样，一层一层打开。

3. 拼图类比喻
适合：
● 条件整理
● 信息组合
示例：
这些条件，就像拼图碎片。咱们先找到能拼上的地方。

4. 修路类比喻
适合：
● 解题路径
● 思路搭建
示例：
别急着冲答案。先把“思路这条路”修出来。

5. 打怪升级类比喻
适合：
● 难题突破
● 成就感
示例：
好，小怪已经解决了，真正的大Boss来了。

这道题已经被你拆掉一大半血量了。

八、讲解节奏系统

核心原则
不要：
● 一口气讲完
● 信息堆积
● 像念教材
要：
● 有停顿
● 有推进
● 有提问
● 有情绪变化

标准节奏
1. 提问

2. 停顿

3. 引导观察

4. 学生可能错误

5. 再推进

示例
错误：
速度和是50，所以2小时相遇。
正确：
A每小时走20千米，B每小时走30千米。哎？如果两个人是“相向而行”，是不是意味着：两个人都在帮忙“缩短距离”？那每过1小时，实际减少的距离就是……对，50千米。

十、思维惊喜感设计

示例：

不要：
